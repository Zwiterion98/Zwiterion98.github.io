Tiempo

Inversion

El ticket promedio de los clientes es una unidad de medida para establecer si estamos frente a un cliente A, B o C. Mediremos las distintas variables tomando como punto de partida este ticket promedio.


Cliente aspiracional

El cliente aspiracional es aquel que cumple con determinados requisitos con base en lo que sería el ideal de la compañía.








Cliente ideal

El cliente ideal es aquel que cumple con los requerimientos para una buena sinergia entre cliente - agencia. Este tipo de clientes facilita la estrategia para llegar más rápido a los objetivos. Además permite un buen trabajo en conjunto.
